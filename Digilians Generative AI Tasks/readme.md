## 📌 Overview

This repository contains practical **Generative AI tasks and experiments** completed as part of the **Digilians Initiative**. It focuses on applying modern AI concepts using Python and Large Language Models (LLMs).

The repository serves as a continuous learning portfolio covering Prompt Engineering, AI model configuration, decoding strategies, and AI-powered applications.

---

## 🎯 Objectives

* Apply Generative AI concepts in real tasks
* Explore Prompt Engineering techniques
* Understand LLM behavior and decoding strategies
* Build AI-powered applications using Python
* Develop practical experience with modern AI tools

---

## 🧠 Topics Covered

* Large Language Models (LLMs)
* Prompt Engineering
* Decoding Strategies
* Temperature, Top-k, and Top-p Sampling
* Token Management & Stop Sequences
* AI APIs & Model Configuration

---
